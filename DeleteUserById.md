# Method :: [DELETE]/Users/DeleteUserById/[id]

## Назначение:

Удаление пользователя из системы 

## Входные данные:

### Параметры запроса:

```json
{
    "id": int //Идентификатор пользователя
}
```

## Пример вызова:

```bash
curl -L -X DELETE 'http://38.135.55.111:5000/Users/1' \
-H 'Session: 2d3c2467-b35e-4768-8e1f-d926f7df76567'
```

## Пример ответа:

```bash
204
```

## Алгоритм:

1. Проверить наличие и валидность хэдера session в запросе. В случае неудачи - вернуть Unauthorized(401).
2. Проверить доступ к редактированию указанного пользователя текущим пользователем, путем проверки наличия роли суперпользователя у редактирующего пользователя или идентичность его идентификатора с редактируемым идентификатором). Если доступа нет - вернуть Forbidden(403).
3. Проверить наличие информации о пользователя в БД. Если запись не найдена - вернуть NotFound(404).
4. Удалить пользователя из БД, а так же связанные с пользователем записи в таблицах user_roles и user_sessions.
7. Вернуть NoContent(204).
