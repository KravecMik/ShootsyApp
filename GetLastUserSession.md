# Method :: [GET]/Users/GetLastUserSession/[id]/session-last

## Назначение:

Получение крайней сессии из БД по идентификатору пользователя 

## Входные данные:

### Параметры запроса:

```json
{
    "id": int //Идентификатор пользователя
}
```

## Выходные данные:
```json
"string": //Сессия пользователя
```

## Пример вызова:

```bash
curl -L -X GET 'http://38.135.55.111:5000/users/2/last-session'
```

## Пример ответа:

#### Статус код:
```bash
200
```

#### Тело ответа:

```bash
"f6360d84-4a16-430d-a5eb-e54d01bca697"
```

## Алгоритм:

1. Получить из БД последнюю существующую сессию пользователя по идентификатору. Если сессии не существует - вернуть NotFound(404).
2. Вернуть информацию о последней сессии пользователя со статусом OK(200).
