# Method :: [POST]/Messages/send/[userLogin]

## Назначение:
**Отправить сообщение в Kafka по логину пользователя**

### Аутентификация
Требуется авторизация

### Параметры запроса
| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|-----------|
| userLogin | string | Да | Логин пользователя-получателя |

### Пример запроса:
Тело запроса: application/json

```json
{
   "Message": "Привет! Как дела?"
}
```


### Логика работы
1. Проверка существования пользователя-получателя по логину
2. Получение ID текущего авторизованного пользователя
3. Отправка сообщения через Kafka Producer
4. Возврат статуса 200 OK при успешной отправке

### Ответы

| Код | Описание |
|-----|-----------|
| 200 OK | Сообщение успешно отправлено |
| 404 Not Found | Пользователь-получатель не найден |
| 401 Unauthorized | Пользователь не авторизован |
