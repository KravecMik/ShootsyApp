# Method :: [POST]/Posts/[postId]/comments

**Добавить комментарий к публикации**

## Аутентификация
Требуется авторизация 

## Параметры запроса

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|-----------|
| postId | int | Да | ID публикации |

### Пример запроса:
Тело запроса: application/json

```json
{
  "Text": "Твой пост очень классный!"
}
```

### Логика работы
1. Получение ID текущего авторизованного пользователя
3. Возврат статуса 201 Created при успешном создании

## Ответы

| Код | Описание | 
|-----|-----------|
| 201 Created | Комментарий успешно добавлен |
| 404 NotFound | Публикация не найдена |
| 401 Unauthorized | Пользователь не авторизован |

## Пример ответа (201 Created)
```json
456
```
