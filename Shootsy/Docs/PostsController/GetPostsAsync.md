# Method :: [GET]/Posts/

**Получить список карточек публикаций**

## Аутентификация
Требуется авторизация 

## Параметры запроса

| Параметр | Тип | Обязательный | Описание | По умолчанию |
|----------|-----|--------------|-----------|-------------|
| Page | int | Нет | Номер страницы | 1 |
| PageSize | int | Нет | Размер страницы | 10 |
| UserId | int | Нет | Фильтр по автору | - |
| Text | string | Нет | Поиск по тексту | - |

## Ответы

| Код | Описание | 
|-----|-----------|
| 200 OK | Список публикаций | 
| 401 Unauthorized | Пользователь не авторизован |
