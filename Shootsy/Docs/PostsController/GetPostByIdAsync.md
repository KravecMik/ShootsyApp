# GET /Posts/[postId]

**Получение карточки публикации по идентификатору**

## Аутентификация
Требуется авторизация

## Параметры запроса

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|-----------|
| postId | int | Да | ID публикации |

## Ответы

| Код | Описание |
|-----|-----------|
| 200 OK | Публикация найдена |
| 404 NotFound | Публикация не найдена |
| 401 Unauthorized | Пользователь не авторизован |

## Пример ответа (200 OK)
```json
{
  "id": 123,
  "createDate": "2024-01-15T10:30:00Z",
  "editDate": "2024-01-15T10:30:00Z",
  "userId": 456,
  "text": "Это моя первая публикация в социальной сети!"
}
```
