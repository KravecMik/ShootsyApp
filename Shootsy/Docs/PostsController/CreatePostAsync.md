# POST /Posts

**Создать публикацию пользователя**

## Аутентификация
Требуется авторизация

## Параметры запроса
**Тело запроса:** `application/json`

| Параметр | Тип | Обязательный | Описание | Ограничения |
|----------|-----|--------------|-----------|-------------|
| Text | string | Да | Текст публикации | Min: 10, Max: 250 символов |

## Пример запроса
```json
{
  "Text": "Это моя первая публикация в социальной сети!"
}
```

| Код | Описание | 
|-----|-----------|
| 201 Created | Публикация успешно создана |
|400 BadRequest|	Ошибка валидации данных	|
| 404 NotFound | Публикация не найдена |
| 401 Unauthorized | Пользователь не авторизован |


## Пример ответа (201 Created)
```json
165456
```
