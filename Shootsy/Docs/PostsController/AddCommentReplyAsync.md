# POST /Posts/[postId]/comments/[commentId]

**Добавить комментарий на другой комментарий к публикации**

## Аутентификация
Требуется авторизация

## Параметры запроса
| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|-----------|
| postId | int | Да | ID публикации |
| commentId | int | Да | ID родительского комментария |

### Пример запроса:
Тело запроса: application/json

```json
{
  "Text": "Твой пост очень классный!"
}
```

## Ответы

| Код | Описание |
|-----|-----------|
| 201 Created | Ответ на комментарий успешно добавлен | 
| 404 Not Found | Публикация или комментарий не найдены | 
| 401 Unauthorized | Пользователь не авторизован |

## Пример ответа (201 Created)
```json
23
```


## Особенности
- Создает древовидную структуру комментариев
