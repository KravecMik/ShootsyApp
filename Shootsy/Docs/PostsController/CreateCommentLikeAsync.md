# Method :: [POST]/Posts/[postId]/comments/[commentId]/likes

**Добавить лайк к комментарию**

## Аутентификация
Требуется авторизация 

## Параметры запроса

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|-----------|
| postId | int | Да | ID публикации |
| commentId | int | Да | ID комментария |

## Ответы

| Код | Описание |
|-----|-----------|
| 201 Created | Лайк успешно добавлен |
| 404 NotFound | Комментарий не найден |
| 400 BadRequest | Лайк уже был добавлен |
| 401 Unauthorized | Пользователь не авторизован |

## Пример ответа (201 Created)

2
