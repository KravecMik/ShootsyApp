# GET /Posts/[postId]/comments/[commentId]

**Получить комментарий по идентификатору**

## Аутентификация
Требуется авторизация

## Параметры запроса

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|-----------|
| postId | int | Да | ID публикации |
| commentId | int | Да | ID комментария |

## Ответы

| Код | Описание | 
|-----|-----------|
| 200 OK | Комментарий найден |
| 404 NotFound | Комментарий не найден |
| 401 Unauthorized | Пользователь не авторизован |

## Пример ответа (200 OK)
```json
{
  "id": 1,
  "createDate": "2025-11-19T04:33:45.473098Z",
  "editDate": "2025-11-19T04:33:45.473162Z",
  "text": "Ну ваще тема!",
  "userId": 1,
  "parentCommentId": null,
  "postId": 1
}
```
