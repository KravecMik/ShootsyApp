# Method :: [POST]/Posts/[postId]/likes

**Добавить лайк к публикации**

## Аутентификация
Требуется авторизация 

## Параметры запроса

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|-----------|
| postId | int | Да | ID публикации |

## Ответы

| Код | Описание | 
|-----|-----------|
| 201 Created | Лайк успешно добавлен | 
| 404 Not Found | Публикация не найдена | 
| 400 Bad Request | Лайк уже был добавлен |
| 401 Unauthorized | Пользователь не авторизован |

## Пример ответа (201 Created)
```json
789
```
