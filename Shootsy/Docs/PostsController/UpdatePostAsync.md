# PATCH /Posts/[postId]

**Обновить карточку публикации**

## Аутентификация
Требуется авторизация

## Параметры запроса

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|-----------|
| postId | int | Да | ID публикации |

## Ограничения
- `UserId` - запрещено к редактированию

| Код | Описание |
|-----|-----------|
| 204 NoContent | Данные успешно обновлены |
| 404 NotFound | Публикация по указанному идентификатору не найдена |
| 401 Unauthorized | Пользователь не авторизован |

## Пример запроса
Тело запроса: `application/json-patch+json`
```json
[
  {
    "op": "replace",
    "path": "/Text",
    "value": "Обновленный текст публикации"
  }
]
```
