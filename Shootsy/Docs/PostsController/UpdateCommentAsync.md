# Method :: [PATCH]/Posts/[postId]/comments/[commentId]

**Обновить комментарий**

## Аутентификация
Требуется авторизация

## Параметры запроса

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|-----------|
| postId | int | Да | ID публикации |
| commentId | int | Да | ID комментария |

**Тело запроса:** `application/json-patch+json`

## Ограничения
- `UserId` - запрещено к редактированию
- `PostId` - запрещено к редактированию  
- `ParentCommentId` - запрещено к редактированию

## Ответы

| Код | Описание |
|-----|-----------|
| 204 NoContent | Комментарий успешно обновлен |
| 404 NotFound | Комментарий не найден |
| 400 BadRequest | Ошибка валидации |
| 401 Unauthorized | Пользователь не авторизован |
|403 Forbidden | Доступ запрещен |
| 500 Internal Server Error | Ошибка при сохранении |
