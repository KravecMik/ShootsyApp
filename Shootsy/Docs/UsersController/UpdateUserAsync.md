# Method :: [PATCH]/Users/[userId]

## Назначение:
Обновление данных пользователя по идентификатору

### Аутентификация:
Требуется авторизация

### Описание параметров запроса:
|Параметр	|Тип	|Обязательный	|Описание|
|:---------:|:---------:|:----------:  |
|userId	|int	|Да	|ID пользователя|

### Пример запроса:
Тело запроса: `application/json-patch+json`

```json
[
  {
    "op": "replace",
    "path": "/firstname",
    "value": "Новое имя"
  },
  {
    "op": "replace", 
    "path": "/description",
    "value": "Новое описание"
  }
]
```

### Ограничения:
* Поле Login недоступно для редактирования

### Возможные коды ответов:
|Код|	Описание|	 
|:---------:|:---------:|
|204 NoContent |	Данные успешно обновлены|
|404 NotFound |	Пользователь не найден|
|400 BadRequest |Ошибка валидации|
| 401 Unauthorized | Пользователь не авторизован |
|403 Forbidden | Доступ запрещен |
