# POST /Files

**Создание карточки файла**

## Аутентификация
Требуется авторизация

## Параметры запроса
**Content-Type:** `multipart/form-data`

| Параметр | Тип | Обязательный | Описание | Ограничения |
|----------|-----|--------------|-----------|-------------|
| File | IFormFile | Да | Загружаемый файл | Только .jpg, .png |

## Логика работы
1. Проверка авторизации и получение ID текущего пользователя
2. Валидация расширения файла (только .jpg, .png)
3. Генерация уникального ключа для объектного хранилища
4. Загрузка файла в MinIO
5. Создание записи в MongoDB с метаданными файла
6. Возврат ID созданной карточки файла

## Ответы

| Код | Описание |
|-----|-----------|
| 201 Created | Файл успешно загружен |
| 400 BadRequest | Неподдерживаемый формат файла |
| 401 Unauthorized | Пользователь не авторизован |
