# DELETE /Files/linked-user/[userId]

**Удалить все файлы по идентификатору пользователя**

## Аутентификация
Требуется авторизация

## Параметры запроса
| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|-----------|
| userId | int | Да | ID пользователя |

## Логика работы
1. Проверка существования пользователя
2. Получение списка всех файлов пользователя
3. Удаление каждого файла из MinIO
4. Удаление всех карточек файлов из MongoDB
5. Возврат статуса 204 NoContent

## Ответы

| Код | Описание |
|-----|-----------|
| 204 NoContent | Все файлы пользователя удалены |
| 404 NotFound | Пользователь не найден |

## Особенности
- Операция массового удаления
- Удаляются как файлы из хранилища, так и метаданные из БД
- Используется для очистки всех файлов пользователя
